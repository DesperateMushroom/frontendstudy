## null / undefined
> null和undefined 都代表的是没有

- null：意料之中 （一般都是开始不知道值，我们手动先设置为null，后期再给予复制操作

```
let num = null;
// 一般最好用null作为初始的空值，因为零不是空值，他在栈内存中有自己的存储空间（占了位置
...
num = 12;
```


- undefined: 意料之外 （不是我能决定的）
```
let num; // 创建变量，没有赋值，默认值就是undefined
...
num = 12;
```
